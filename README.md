# Blog Preview Card

Este repositório contém a solução para o desafio "Blog preview card" do Frontend Mentor.

## O que está aqui

- `index.html` — página principal do projeto
- `css/styles.css` — estilos utilizados para o layout e responsividade
- `assets/` — recursos do projeto, incluindo imagens e fontes

## Como visualizar

1. Abra `index.html` no navegador
2. Ou use uma extensão como Live Server para abrir o projeto localmente

## Sobre o projeto

A página reproduz um cartão de prévia de blog com:

- layout responsivo
- estilo inspirado no design do Frontend Mentor
- hover e foco para elementos interativos
